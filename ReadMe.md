# Sugar & Butter Bakery 🍰🍞

Welcome to the **Sugar & Butter Bakery** web application! This is a fully functional e-commerce / management platform designed for a modern bakery. Customers can browse delicious treats, place orders, and the admin can manage inventory and sales.

---

## 🚀 Features

*   **User Authentication:** Secure registration and login for customers and admins.
*   **Product Catalog:** Browse items by categories (Cakes, Pastries, Breads, Cookies).
*   **Shopping Cart & Checkout:** Easy add-to-cart functionality with a smooth checkout process.
*   **Order Tracking:** Customers can view their order history and current status.
*   **Admin Dashboard:** Manage products (CRUD operations), view orders, and track daily sales.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
